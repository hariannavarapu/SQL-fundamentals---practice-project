# SQL Fundamentals – Practice Project

This repository demonstrates core SQL skills using a small sample database.

## Topics Covered
- **DDL** – Database and table creation
- **DML** – Insert, update and delete operations
- **DQL** – Queries using filters, joins, subqueries and aggregations

## Tools Used
- SQL (MySQL / SQL Server compatible)

## Purpose
Created as part of my learning journey to build a strong foundation in SQL for entry-level data analyst roles.
